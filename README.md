# bib_creation
This code provides a basic tool to created personnalized bibs for your race in an automated way using Python. This is definitely not a robust and efficient tool but it successfully helped me in organizing a race (and playing with sphinx) when it was developped.  

Ce code fournit un outil basique pour concevoir des dossards personnalisés de manière automatique pour votre course en utlisant le langage Python. Sans être robuste et aboutit, cet outil a pu être utilisé dans l'organisation d'une course lorsqu'il a été développé. Et a été un prétexte pour jouer avec Sphinx à l'époque!
